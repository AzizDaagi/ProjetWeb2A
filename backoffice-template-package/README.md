# Backoffice Template Package

Package de partage du backoffice Smart Nutrition.

## Contenu
- `template.php` : point d'entree de la template
- `includes/header.php` : debut HTML + chargement CSS
- `includes/admin_nav.php` : barre laterale et topbar admin
- `includes/footer.php` : fin HTML + chargement JS
- `pages/dashboard.php` : page dashboard exemple
- `pages/users-list.php` : page liste utilisateurs exemple
- `assets/backoffice.css` : styles du backoffice
- `assets/backoffice.js` : interactions JS du backoffice

## Utilisation
Ouvrir `template.php` dans un environnement PHP local.
